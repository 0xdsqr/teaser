Teaser components are designed to be composable. Mix and match them to create exactly what you need.

## Component Composition

Each Teaser component is self-contained and can be used independently or combined with others.

```tsx
import {
  Teaser,
  TeaserIcon,
  TeaserHeader,
  TeaserTitle,
  TeaserDescription,
  TeaserForm,
  TeaserInput,
  TeaserButton,
} from "@/components/ui/teaser";

export function CustomLayout() {
  return (
    <Teaser className="max-w-lg">
      <TeaserIcon animated>ðŸŽ‰</TeaserIcon>
      <TeaserHeader>
        <TeaserTitle>Custom Title</TeaserTitle>
        <TeaserDescription>Your custom description</TeaserDescription>
      </TeaserHeader>
      <TeaserForm onEmailSubmit={handleSubmit}>
        <TeaserInput />
        <TeaserButton>Subscribe</TeaserButton>
      </TeaserForm>
    </Teaser>
  );
}
```
