Check out these examples to see Teaser in action.

## Kitchen Sink

The complete example with all Teaser features:

```bash
cd examples/kitchen-sink
bun install
bun run dev
```

Visit `http://localhost:3000` to see the demo.

## Dark Mode

Teaser automatically supports dark mode:

```tsx
<html className="dark">
  <body>
    <Teaser>{/* Your content */}</Teaser>
  </body>
</html>
```
