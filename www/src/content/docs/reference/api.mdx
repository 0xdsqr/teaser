Complete reference for all Teaser components and their props.

## Teaser

Root container component.

```tsx
<Teaser className="max-w-md">{/* Content */}</Teaser>
```

### Props

| Prop        | Type      | Default |
| ----------- | --------- | ------- |
| `children`  | ReactNode | -       |
| `className` | string    | -       |

## TeaserHeader

```tsx

  <TeaserTitle>Title</TeaserTitle>
  <TeaserDescription>Description</TeaserDescription>
</TeaserHeader>
```

## TeaserForm

Email form with validation.

```tsx
<TeaserForm onEmailSubmit={(email) => console.log(email)}>
  <TeaserInput />
  <TeaserButton>Submit</TeaserButton>
</TeaserForm>
```

### Props

| Prop            | Type                    | Default |
| --------------- | ----------------------- | ------- |
| `onEmailSubmit` | (email: string) => void | -       |
| `children`      | ReactNode               | -       |
